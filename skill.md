# 技能

## 项目目标

开发一个订单卖车的TOB后台Saas服务系统

## 项目背景

随着移动互联网的普及，越来越多的人选择在App上购买汽车。然而，传统的汽车销售模式已经无法满足客户的需求。为了满足客户的需求，开发一个订单卖车的TOB后台Saas服务系统是非常必要的。

## 技术栈

react + node.js

## 功能模块

1.  用户登录模块
    - 用户注册
    - 用户登录
    - 用户权限管理
2.  车辆管理模块
    - 车辆信息录入
    - 车辆信息查询
    - 车辆状态管理
3.  订单管理模块
    - 订单创建
    - 订单查询
    - 订单状态管理
4.  基础数据模块(比如车辆品牌、车辆型号、车辆颜色、车辆状态等)
    - 品牌管理
    - 型号管理
    - 颜色管理
    - 状态管理
5.  支付模块
    - 支付订单
    - 查询支付订单状态

# 后端设计
  1. 数据库设计
     - 用户表
     - 车辆表
     - 订单表
     - 支付表
     - 基础数据表(品牌、型号、颜色、状态等)

  2. API设计 restful风格
     - 用户登录注册API
     - 车辆管理API
     - 订单管理API
     - 支付管理API
     - 基础数据API(品牌、型号、颜色、状态等)
  3. 架构设计
     - 后端架构采用MVC模式
     - 数据库采用MySQL
     - 单体架构
# 前端设计
    - 前端采用react框架
    - 前端路由采用react-router-dom
    - 前端状态管理采用redux
    - 前端UI组件库采用ant-design
    - 前端部署采用nginx
